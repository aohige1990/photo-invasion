この端末では全画面にできません

iPhoneは本体側の制限で、ブラウザから全画面にする機能が使えません(Safari・Chromeなどブラウザを変えても同じです)。

かわりにホーム画面に追加すると、アドレスバーのない全画面で遊べます。

やり方(Safariの場合)
1. 画面下の「共有」ボタン(□に↑のマーク)を押す
2. 「ホーム画面に追加」を選ぶ
3. 「追加」を押す
4. ホーム画面にできたアイコンから起動する

起動したあと端末を横向きにすると、いちばん広く表示されます。

🔰 説明書

このゲームは?

好きな写真を主人公や敵の顔にはめこんで遊べる、横スクロールのシューティングゲームです。写真はあなたの端末の中だけで処理され、インターネットに送られることはありません。

まず操作を覚えよう

  • ◀ ▶ … 左右に歩く
  • ショット … メイン武器を撃つ(押しっぱなしで連射)
  • サブ … サブ武器を使う(パンチや刀など)
  • 武器切替 … 持っている武器を持ちかえる
  • 画面右上の II で一時停止、🔊 で音のオンオフ
  • ハートが体力です。敵にぶつかると減り、0になるとゲームオーバーです

🎮 ステージモード

STAGE 1から順番に進んでいく、このゲームのメインモードです。

  • 決められた数の敵を倒すとステージクリアです(画面右上に「3/20」のように出ます)
  • 5の倍数のステージ(5、10、15…)はボス戦です。大きな敵が1体だけ出てくるので、倒せばクリアです
  • 敵を倒すとコイン($)がもらえます。コインはショップで武器を買うのに使います
  • クリアしたステージは記録されるので、次からはそこから始められます
  • やられてしまっても、そのステージでためたコインはもらえます。何回でも挑戦できます

♾️ エンドレス

倒れるまでひたすら敵と戦い続けるモードです。どこまで倒せるかの記録に挑戦します。

  • コインはもらえません。ステージモードで買った武器も使えません
  • かわりにハンドガンとパンチだけの状態からスタートし、遊びながら武器が増えていきます
  • 敵を10体倒すごとにWAVEが1つ進みます
  • ときどきボスが出てきます。倒すと新しい武器がもらえます(ガトリングガン→ナイフ→ショットガン…の順)
  • すべての武器がそろったあとも、5WAVEごとに「強敵」が出てきます
  • 倒された時の撃破数がランキングに残ります(上位5件)

🛒 ショップ

ステージモードでためたコインで武器を買ったり強くしたりする場所です。

  • メインとサブの2種類のタブがあります
  • 武器は安い順に解放されます。今の武器をレベル5まで上げると、次の武器が買えるようになります
  • 買った武器はレベルアップできます。1レベルごとに威力が25%上がり、最大レベル5です
  • 装備ボタンを押すと、その武器を使うようになります
  • メイン武器をはじめて買った時だけミニゲームが始まります(ボーナスコインがもらえます)

メイン武器の順番: ハンドガン → ガトリングガン → ショットガン → ロケットランチャー → ビームガン → レールガン

📷 キャラ設定と切り抜き方

写真をキャラクターにはめこむ画面です。主人公・敵エイリアン①②③・ボス・ミニゲーム用キャラ①〜⑥を設定できます。

2つのモードがあります

  • 顔はめ込み … 写真の真ん中を丸く切りぬいて、キャラの頭にはめます。体はゲームのイラストのままです。かんたんなのでこちらがおすすめです
  • 全身画像 … 写真そのものがキャラになります。選ぶと背景の切り抜き画面が開きます

背景の切り抜きのやり方

  1. 「全身画像」を選んでから「画像フォルダから選ぶ」で写真を選びます
  2. 切り抜き画面が開いたら、消したい背景の部分を指でタップします。タップした色とつながっている似た色が消えます
  3. 消えかたが足りない時は「削除の強さ」を右に動かして、もう一度タップします(強くしすぎると人物まで消えるので少しずつ)
  4. すみに残った背景は「消しゴム」でなぞって消します
  5. 消しすぎたら「復元」でなぞると元に戻ります
  6. 「移動」にすると、指1本で動かす・指2本で拡大縮小ができます。細かい所を消す時に便利です
  7. 失敗したら「元に戻す」、全部やり直すなら「最初から」を押します
  8. 右下の小さなプレビューでゲーム内の見え方を確認できます
  9. よければ「この画像を使用」を押して完了です

コツ: 背景が白い壁など、1色でシンプルな写真がいちばんキレイに切り抜けます。うしろがゴチャゴチャした写真は苦手です。市松模様(チェック柄)になっている所が「透明になった部分」です。

⚠️ 写真はブラウザを閉じると消えます。遊ぶたびに設定し直してください(写真だけは保存されない仕組みです)。

💾 データのバックアップ

コインや武器、どこまで進んだかを文字コードにして保存できます。機種変更した時や、まちがって消してしまった時に元に戻せます。

保存のやり方

  1. 「最新のコードを生成」を押します
  2. 「コピーする」を押します
  3. そのコードをLINEのメモやメールなど、自分だけが見られる所に貼り付けて保管します

元に戻すやり方

  1. 保管しておいたコードをコピーします
  2. 「クリップボードから貼り付け」を押します(うまくいかない時は入力らんを長押しして貼り付け)
  3. 「このコードで復元する」を押します

⚠️ 復元すると今の進み具合は上書きされて消えます。先に今のコードを保存しておくと安心です。写真とミニゲームの記録はバックアップに入りません。

🎯 ミニゲーム

5種類あります。メイン武器をはじめて買うと解放され、そのあとはタイトルの「ミニゲーム」からいつでも遊べます。クリアするとボーナスコインがもらえます。

  • 顔ハメもぐら叩き(ガトリングガンで解放) … 穴から出てきた顔を20秒間タップ。叩くほど顔がボコボコになります
  • 顔ハメ神経衰弱(ショットガンで解放) … 12枚のカードから同じ顔のペアを探します。45秒
  • 顔ハメ福笑い(ロケットランチャーで解放) … 眉・目・鼻・口をドラッグして顔を完成させます。正しい位置に近いほど高得点
  • 顔ハメだるまさんがころんだ(ビームガンで解放) … 「進む」を連打してゴールを目指します。ただし振り向いた瞬間に押すと失敗。振り向くアニメをよく見て
  • 顔ハメ横スクロール(レールガンで解放) … タップでジャンプして走り続け、距離をきそいます

横スクロールのくわしいルール

  • 下の障害物(段ボール・パレット) … ジャンプで飛びこえます
  • 上の障害物(黄色と黒のバー) … ジャンプすると頭をぶつけます。ジャンプせずに走りぬけるのが正解です
  • 拾えるアイテム … 安全靴(パレットを1回防ぐ)、ヘルメット(上のバーを1回防ぐ)、台車(段ボールを3回)、フォークリフト(パレットを3回)、ハンディ端末(100m飛ぶ・レア)
  • 持っているアイテムは画面の左はしに表示されます

それぞれのミニゲームで自己ベストが記録され、一覧に表示されます。

⛶ 全画面について

画面の右上にある ⛶ を押すと、アドレスバーが消えて画面いっぱいに広がります。もう一度押すと元に戻ります。

  • Android … ⛶ を押すだけでOKです。対応している端末なら自動で横向きにもなります
  • PC … ⛶ を押すと全画面になります
  • iPhone … 残念ながらiPhone本体の仕様で、ブラウザから全画面にすることができません(Safariでも、Chromeなど他のブラウザに変えても同じです)

iPhoneで全画面にしたい場合

  1. Safariでこのゲームを開きます
  2. 画面下の「共有」ボタン(□に↑のマーク)を押します
  3. 「ホーム画面に追加」を選んで「追加」を押します
  4. ホーム画面にできたアイコンから起動すると、アドレスバーのない全画面で遊べます

横向きにするともっと広く遊べます。端末の「画面の向きのロック」がオンだと横向きにならないので、その時はロックを解除してください。

困った時は

  • 写真が消えた … ブラウザを閉じると写真は消える仕組みです。もう一度設定してください
  • 画面が古いまま … ページを下に引っぱって更新するか、ブラウザのキャッシュを消してください
  • コードで復元できない … コードの一部が欠けていることが多いです。アプリの「コピーする」ボタンでコピーしたものを使ってください
